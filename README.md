# 취향을 공간에 풀어내는 시대

ELURA Culture Forecast 01의 한국어 웹 ebook입니다. 취향, 집, AI, 주문형 생산이 만나는 변화를 29개 장과 215개의 디지털 페이지로 구성했습니다.

## 독서 기능

- 데스크톱 연속 읽기, 목차 검색, 글자 크기, 다크 모드
- 모바일 페이지 읽기, 좌우 탭과 가로 스와이프, 키보드 이전/다음
- 모바일 오른쪽 가장자리 세로 제스처 및 슬라이더 밝기 조절
- 페이지와 연속 스크롤 모드 전환
- 읽던 페이지, 화면 설정, 워크북 답변의 기기 내 저장
- A5 페이지 단위 인쇄 및 PDF 저장

## 개발

Node.js 22.13 이상이 필요합니다.

```bash
npm install
npm run dev
npm run build
npm test
```
